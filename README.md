## HOW TO PLAY

* "A" Key - Move Left
* "D" Key - Move Right
* "Alt" Key - Shoot (If you got power up)
* "ESC" Key - Pause/Restart Level

## COLLECTIBLES

* (Apple) - Scales you down!
* (Candy) - Scales you up! 
* (Power-up) - Lets you shoot!
* (Key) - Opens doors!
